Projeto 1 - Análise de NDVI Offline

Desenvolvi um projeto qual automatiza a exportação de mapas NDVI real ou aproximado depedendo da disposição das imagem de Drone (RGB - aproximado) ou (NIR + RGB - verdadeiro)

Este projeto parte do Python e automatiza a Analise das imagens de input (ortomosaico ou imagens soltas), neste caso foram imagens soltas, foi um total de 307 imagens analisadas em torno de 20 - 30 minutos, de modo offline.

Qual a Vantagem? Não precisar de um equipamento extremamente caro por poder fazer o calculo aprximado, tendo em consideração de que existem correções para redução do valor de erro se torne minimo.

A analise pode ser feita um ortomosaico gigantesco ou no mosaico quando fragmentado, possívelmente no futuro vou implementar para o código juntar, unir as imagens e exportar o tiff do NDVI.
