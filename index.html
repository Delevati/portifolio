Ferramentas desenvolvidas

"Projeto Codeleaf: Análise Offline Automatizada de Imagens NDVI."

Desenvolvi um projeto inovador em Python que automatiza a exportação de mapas NDVI. Ele funciona tanto com imagens aproximadas quanto com resultados precisos, dependendo do tipo de imagens disponíveis. O projeto é altamente eficiente, processando 307 imagens em apenas 20 a 30 minutos, tudo offline.

Uma das maiores vantagens desse sistema é a economia de custos, pois elimina a necessidade de equipamentos especializados e caros para cálculos aproximados de NDVI. Além disso, oferece métodos de correção que tornam os resultados aproximados ainda mais confiáveis.

O sistema é flexível o suficiente para trabalhar com ortomosaicos de grande escala ou mosaicos fragmentados. Estou planejando atualizações futuras que permitirão consolidar várias imagens em um único arquivo TIFF de NDVI. Resultados completos aqui.
